# Technical Interview Prep

Common data science interview questions - practice until you can solve them without help.

---

## SQL Interview Questions

### Easy
1. **Second Highest Salary** - Find the employee with the 2nd highest salary
2. **Duplicate Emails** - Find all duplicate email addresses
3. **Employees Earning More Than Manager** - Self-join problem

### Medium
4. **Department Top Earner** - Highest paid employee in each department
5. **Consecutive Days** - Find users who logged in 3+ consecutive days
6. **Running Total** - Calculate cumulative sum (window function)

### Hard
7. **Median Salary** - Calculate median (not average)
8. **Year-over-Year Growth** - Compare this year vs last year sales

---

## Python/Pandas Interview Questions

### Data Manipulation
1. Clean a messy CSV with missing values, duplicates, wrong types
2. Merge two DataFrames on multiple keys
3. Pivot data from long to wide format
4. Calculate rolling averages

### Logic Problems
5. FizzBuzz (classic)
6. Find pairs that sum to target
7. Group anagrams together
8. Implement a simple LRU cache

## Statistics Questions

1. Explain p-value to a non-technical person
2. What's the difference between Type I and Type II errors?
3. When would you use median instead of mean?
4. How do you detect outliers?
5. Explain A/B testing process

---

## ML Questions

1. How do you handle imbalanced classes?
2. What's overfitting and how do you prevent it?
3. Explain the bias-variance tradeoff
4. When would you use L1 vs L2 regularization?
5. How do you choose between precision and recall?

---

## Case Study Questions

1. "Customer churn is increasing. How would you investigate?"
2. "We want to predict which users will upgrade to premium. Walk me through your approach."
3. "How would you measure the success of a new feature?"

---

## Practice Workflow

1. Set a timer (20-30 min per problem)
2. Write your solution WITHOUT looking up answers
3. Run it with Claude Code
4. If stuck, ask for hints not answers
5. After solving, ask "Is there a better way?"
