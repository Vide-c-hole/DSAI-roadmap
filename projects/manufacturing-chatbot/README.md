# Manufacturing Defect Analysis Chatbot

> Portfolio Project #1 - Demonstrates domain expertise + LLM skills

## Status: Planning

## Why This Project

- Combines ME background with AI skills
- Shows practical industry application
- Unique positioning vs typical DS candidates

## Tech Stack

- Python + Streamlit
- OpenAI/Claude API for LLM
- ChromaDB for vector storage
- Pandas for data manipulation

## Features (Planned)

1. Upload defect reports/images
2. Query historical defect data
3. Get root cause suggestions
4. Visualize defect patterns

## Project Structure

