# DSAI Learning Companion

> Supporting resources for NTU SCTP Part-Time Data Science & AI course

**Primary:** NTU SCTP course materials
**This repo:** Supplementary notes, cheatsheets, portfolio projects

---

## Structure

```
DSAI-roadmap/
├── knowledge/          # Quick reference & notes
│   ├── concepts/       # Course concept summaries
│   ├── tools/          # Cheatsheets (SQL, Pandas, etc.)
│   └── papers/         # Optional deep dives
├── projects/           # Portfolio projects
│   └── manufacturing-chatbot/
```

## How to Use

| Need | Action |
|------|--------|
| Study new topic | Use NTU course materials |
| Quick reference | Check `knowledge/tools/` cheatsheets |
| Practice problems | Use LeetCode, HackerRank, Kaggle |
| Build portfolio | Work in `projects/` |
